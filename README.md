
## 🎯 프로젝트 개요

### ✅ 문제 정의
- 유튜브 댓글 영역은 **개방형 구조**로 인해 광고/투자 유도/사기 링크 등의 스팸 댓글이 지속적으로 증가합니다.
- 특히 스팸 댓글은 **후기·경험담 형태로 위장**되는 경우가 많아, 일반 사용자가 진위 여부를 빠르게 판단하기 어렵습니다.
- 기존 키워드 기반 필터링은 사용자가 **“이 영상이 얼마나 위험한지”**를 한눈에 파악하기 어렵다는 한계가 있습니다.

### 💡 해결 방안
- 사용자가 **유튜브 영상 링크 하나만 입력**하면, 댓글을 수집·분석하여  
  **광고/스팸/사기성 패턴을 탐지**하고 **요약 리포트(대시보드)**로 제공합니다.
- **YouTube 공식 Data API** 기반으로 수집하여 정책 위반(스크래핑) 없이 안전하게 동작합니다.

## 🧠 핵심 컨셉

- ❌ 단순 댓글 목록/키워드 차단
- ✅ **“URL 입력 → 위험도 요약 → 의심 댓글 근거 제공”** 중심의 분석기(Information Tool)

> “이 영상 댓글, 안전해? 위험해?”  
> 한 번의 입력으로 빠르게 확인하는 **사용자 중심 분석 경험**

## 🧱 기술 스택

### Frontend
- **Next.js 14 (App Router) + TypeScript**
- **TailwindCSS + shadcn/ui** (다크 테마, 카드형 레이아웃, 일관된 UI 컴포넌트)
- **Recharts** (분류 분포/요약 차트)
- **Zustand** (분석 결과/설정/히스토리 상태 관리)

### Backend
- **FastAPI (Python)**
- 댓글 수집 API / 전처리 / 규칙 기반 분석 / 결과 반환

### External API
- **YouTube Data API v3**
  - 공식 API 기반 댓글 수집(스크래핑 X)

### Database (선택)
- MongoDB 또는 SQLite  
  - 분석 결과/사용자 설정/분석 이력 저장

## 🖥️ 서비스 설명

### 1) 랜딩 페이지 (소개)
- 서비스 한 줄 설명 + 핵심 가치 제안
- 신뢰 요소 강조  
  - ✅ YouTube 공식 API 기반  
  - ✅ 스크래핑 없음  
  - ✅ 개인정보 저장 최소화
- 대상 사용자 안내  
  - **시청자:** 시청 전 댓글 위험도 확인  
  - **크리에이터:** 내 영상 댓글 스팸 패턴 요약(확장)

---

### 2) URL 입력 페이지 (분석 시작)
- 큰 입력창 1개 + “분석 시작” 버튼  
- 예시 링크 제공 / Enter 제출 지원  
- 분석 중 로딩 UI(Skeleton/Spinner) 표시

---

### 3) 분석 리포트 페이지 (대시보드)
- 상단: **위험도 상태 요약(SAFE/WARN/RISK)** + 핵심 지표 카드
- 중단: 분류 분포 차트(AD/SPAM/SUS/OK)
- 하단: 의심 댓글 리스트(탐지 사유/점수/패턴 근거 제공)
